# Pizza_times_bot 
 
Бот который помогает выбрать пиццу  


## Список команд  

 * /start - Начало работы  
 * /help - список команд  
 * /friend - реферальная ссылка  
 * /sale - скидка  
 * /call - можно нам позвонить  
 * /geophone - Ваш телефон или локация
